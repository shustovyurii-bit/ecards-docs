# Статистика по дням

Раздел **Статистика и отчёты → Статистика трат по картам**.

Расходы в разбивке EUR / USD по дням, неделям и месяцам.

Фильтры: карты, период, счёт, держатели карт, автор операции (в т.ч. можно выбрать себя), команды, группа.

*Статистика по дням отображается с учётом GMT+03:00.*
