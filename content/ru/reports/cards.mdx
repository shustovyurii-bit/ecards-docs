# Отчёты по картам

Раздел **Виртуальные карты → Операции по картам**.

Отображаются все операции по картам — авторизации, деклайны, возвраты (см. [статусы](#term-статусы-операций-по-картам)).

![Операции по картам — список с фильтрами](/images/screenshots/cards-operations.jpg)

**Как сформировать отчёт:**
1. Перейдите в **Виртуальные карты → Операции по картам**.
2. Выставьте нужные фильтры.
3. Нажмите кнопку **«Сформировать отчёт»** в правом верхнем углу.
4. Готовый отчёт появится в разделе **Уведомления** — оттуда его можно скачать.

**Доступные фильтры:** карта, группа, счёт, держатель карты, автор операции, период, статус операции (в т.ч. Declined (Штраф) — для подсчёта суммы штрафов).

**Форматы выгрузки:** CSV или XLSX.

*Рекомендация: CSV удобнее открывать через Google Таблицы — Excel может некорректно форматировать номера карт и CVV. Если используете Excel, после открытия файла вручную установите формат столбцов «Текст».*

**Q: Карту перенесли из одной группы в другую — в каком отчёте появятся её операции?**

Отчёт выгружается по операциям карты, а не по группе. Если карту перенесли в другую группу — все её операции будут отображаться при фильтрации по текущей группе (той, в которой карта находится на момент выгрузки).
