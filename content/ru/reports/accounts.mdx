# Отчёты по счетам

Раздел **Мои счета → Операции по счетам**.

Отображаются переводы, изменения лимитов, штрафы и системные списания (см. [типы](#term-статусы-операций-по-счетам)).

Доступна кнопка **«Сформировать отчёт»** для экспорта.
