# Счета с общим балансом

[Счёт с общим балансом](#term-счета-с-общим-балансом) — все карты используют общий баланс счёта без индивидуальных лимитов. При оплате денежные средства списываются напрямую со счёта, без предварительного пополнения карты.

**Q: Для каких карт доступен?**

Доступен для всех BIN-ов. Карты с общим балансом можно приобрести в разделе **Покупка карт** → вкладка **С общим балансом**.
